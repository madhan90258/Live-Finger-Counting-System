# Live Finger Counting System

A real-time computer vision application that detects and counts the number of fingers shown to a webcam using **Python**, **OpenCV**, and **MediaPipe**.

## 📌 Project Overview

This project captures live video from a webcam and uses MediaPipe's hand landmark detection model to identify hand positions and count the number of raised fingers in real time.

It serves as an introduction to computer vision, hand tracking, gesture recognition, and real-time human-computer interaction.

## 🚀 Features

* Real-time webcam video processing
* Hand detection and tracking
* Finger counting using hand landmarks
* Fast and accurate performance
* Simple and beginner-friendly implementation

## 🛠️ Technologies Used

* Python
* OpenCV
* MediaPipe Hands API

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/madhan90258/Live-Finger-Counting-System.git
```

2. Navigate to the project directory:

```bash
cd Live-Finger-Counting-System
```

3. Install the required dependencies:

```bash
pip install opencv-python mediapipe
```

## ▶️ Usage

Run the Python script:

```bash
python Live-Finger-Counting-System.py
```

The webcam will open, and the application will detect and count the number of fingers shown in real time.
